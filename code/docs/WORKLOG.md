| 2026-04-22 | **LC-conditioned eigenvalue PINN (08_laplace_1d_lc).** Replaced the K independent Rayleigh networks from Apr 21 with a single `LCEigenmodeNet` conditioned on mode index k. Loss = Σ_k (1/k) · R(u(·, k)) + α · Σ_{i<j} cos²(u(·, i), u(·, j)) — Ky Fan variational principle with cos² orthogonality (same scale-invariance fix as Apr 21). Three design choices that all mattered: (1) one-hot k encoding (DIM_LAMBDA=K=5) — scalar/normalised k encodings collapsed adjacent slots; (2) curriculum, phase j uses only slots 1..j in the loss (mirrors sequential deflation with shared weights); (3) cos² penalty. Results at K=5, 100k Adam, [64,64,64,64] trunk (~13k params — same as 5 separate 32-32-32 nets combined): eigenvalues recovered to ≤5% (beats sequential on 4/5 modes); eigenfunctions competitive for k=1..3 (rel-L2 2–7%), degrade to ~28% for k=4, 5 due to shared-trunk mode mixing. Ordering check passes (slot k → k-th Rayleigh rank). Paper framing: extension direction, not main result — generalises LC-PINN's scope from "condition on loss weights" to "condition on any problem parameter". Burgers 357× + λ-invariance remain the paper spine. Files: `pinns/equations/laplace_1d_lc.py`, `scripts/run_laplace_1d_lc.py`, `notebooks/08_laplace_1d_lc.ipynb`, checkpoint `laplace_1d_lc.pt`. |

| 2026-04-22 | **Advisor Apr 21 ask: Laplace 1D eigenvalue variational PINN.** Built `pinns/equations/laplace_1d.py` + `notebooks/06_laplace_1d.ipynb`: hard Dirichlet BC via `B(x) = x(π−x)`, Rayleigh-quotient objective, sequential deflation. First run collapsed modes 2–5 onto mode 1 (λ̂ ≈ 1.04 for every k). Root cause: Rayleigh is scale-invariant, so the standard `(E[u·u_j])²` penalty is escaped by shrinking ‖u‖→0. Fix: `cos²(angle(u, u_j))` direction penalty. After fix: all 5 modes distinct, λ errors ≤1.3% for k=1..4 (k=5: 7.8%, spectral-bias limit of 32-32-32 MLP). Also built `notebooks/07_navier_stokes_2d.ipynb` + `pinns/equations/navier_stokes_2d.py` (Taylor-Green benchmark).
