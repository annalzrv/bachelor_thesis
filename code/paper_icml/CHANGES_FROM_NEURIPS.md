### 3. NEW Section 4 + Theorem 4.1 (advisor's main contribution)

`paper_icml/main.tex` lines 580–767:

> **§4 — Local effect of loss-conditioning on PINN optimization**
>
> Sets up a local quadratic model `L_λ(θ) = ½(θ−θ*)ᵀ H_λ (θ−θ*)`
> around a common optimum `θ*` (well-specified setting where all loss
> components vanish). Models the conditional architecture as
> `θ_eff(λ;θ,η) = θ + B(λ,η)` with bilinear `B(λ,η) = Σ λⱼ Mⱼ η`
> (concat conditioning is exactly this; FiLM is more complex but the
> qualitative result transfers).
>
> **Theorem 4.1 (Local loss-decay advantage of loss-conditioning).**
> Under continuous-time gradient flow, at any common effective
> parameter, the LC parameterisation has instantaneous loss-decay rate
>
> ```
> −d/dt L_λ(θ_eff)  =  −d/dt L_λ(θ_reg) + ‖D_η B(λ,η)ᵀ ∇L_λ‖²,
> ```
>
> i.e. ≥ regular PINN's, with the explicit non-negative gap
> `‖D_η Bᵀ ∇L_λ‖²` from η-direction descent.

### 4. NEW Appendix A — Derivation of the local loss-decay result

`paper_icml/main.tex` lines 1112–1564, with subsections:

- A.1 — Local quadratic model for multi-objective PINN losses.
- A.2 — Effective parameters induced by loss-conditioning (bilinear B
  derived from concat in the first layer).
- A.3 — Proof of Theorem 4.1 (chain rule + gradient flow on the joint
  (θ, η) space; clean and self-contained).
- A.4 — Interpretation (local mechanism, not a global guarantee;
  ‖D_η Bᵀ ∇L_λ‖² vanishes only when the loss gradient is orthogonal to
  the η-subspace).
- A.5 — Remark on discrete gradient descent (continuous-time result
  matches to first order in step size α).
